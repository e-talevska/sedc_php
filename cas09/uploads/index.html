Introduction

What is PHP?

  • PHP == 'PHP: Hypertext Preprocessor'
  • Widely-used open source language, especially suited to generate dynamic web-pages
  • Server side scripting language
  • Interpreted language, scripts are parsed at run-time rather than compiled beforehand

What is a PHP File?

  • PHP files can contain text, HTML, CSS, JavaScript, and PHP code
  • PHP code reside between reserved PHP tags
    • This allows the programmer to embed PHP scripts within HTML pages

    • 
									<p> <?php echo "Hello class!"; ?> </p>

  • PHP code are executed on the server, and the result is returned to the browser as plain HTML
  • PHP files have extension ".php"

What Can PHP Do?

  • Can generate dynamic page content
  • Can create, open, read, write, delete, and close files on the server
  • Can collect form data
  • Can add, delete, modify data in your database
  • Can be used to control user-access

Why choose PHP?

  • Open source, free software
  • Cross platform - Runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
  • Is compatible with almost all servers used today (Apache, IIS, etc.)
  • Compatible with many popular databases (MySQL, Oracle, PostgreSQL, mSQL, DB2 Sybase)
  • Various built-in functions allow for fast development
  • Large, active, developer community
    • Over 20 million web sites use PHP

PHP History

  • Version 1 - 1994
    • Written by Rasmus Lerdorf
    • CGI (Common Gateway Interface) binaries in the C programming language to help him maintain his personal home page
  • Version 2 - 1995
    • First public release
    • Code that interprets data submitted by web form
    • "Personal Home Page Tools"

PHP History (1)

  • Version 3 - 1998
    • Two more developers (Andi Gutmans and Zeev Suraski)
    • PHP was rewritten
    • "PHP: Hypertext Preprocessor"
  • Version 4 - 2000
    • Company "Zend" was formed, which still guides PHP developemnt
    • PHP core was rewritten

PHP History (2)

  • Version 5 - 2004
    • New features
    • Better performance
    • Improved Object – Oriented programming
  • Version 6 (Never released)
    • Initiation to bring native Unicode support throughout PHP
    • In 2010, the project was officially abandoned
  • Version 7 - 2015
    • Optimized PHP performance
    • New features
    • Improvements that require backward-compatibility breaks

Variables

Variables

  • In all programming languages we need to keep truck of data
  • Container
  • Giving a name to a piece of computer memory

Variables

  • Data stored in variables can be from different type
  • PHP allows storing of 8 types of variables
    • Four scalar (single-valued) data types:
      • Integer — Used for whole numbers
      • Float (also called double) — Used for real numbers
      • String — Used for strings of characters
      • Boolean — Used for true or false values
    • Two nonscalar (multivalued) data types:
      • Array — Used to store multiple data items
      • Object — Used for storing instances of classes
    • Two special types are also available:
      • Resource They represent external resources(such as database connections)
      • NULL (not been given a value, have been unset, or have been given the specific value NULL)

Variables

  • PHP does not require variables to be declared
  • A variable is created when you first assign a value to it
  • PHP is called weakly typed, or dynamically typed language
    • The type of a variable is determined by the value assigned to it

Variables

  • PHP variables must begin with a $ sign
  • Followed by letter or underscore
  • Can contain letters, numbers or underscores
  • No spaces
  • Case-sensitive ($Foo != $foo != $fOo)

Example

$item
$Item
$myVariable
$this_variable
$this-variable // Not valid
$99problems // Not valid
$product3
$_book // Not recommended 
$__bookPage // Not recommended

Variables

  • Globally and locally-scoped variables
    • Global variables can be used anywhere
    • Local variables restricted to a part of the code (function or class)
  • Certain variable names reserved by PHP (Superglobal variables)
    • Form variables ($_POST, $_GET)
    • Server variables ($_SERVER)
    • More on: php.net/manual/en/reserved.php

String

  • Sequence of characters
  • A string can be any text inside quotes. You can use single or double quotes.
  • Double quotes are used when we want a variable to be interpreted not literally outputed


						//same
$x = "Hello world!";
$y = 'Hello world!';

//not the same
$name = "Emilija";
$x = "Hello $name!"; //Hello Emilija!
$y = 'Hello $name!'; //Hello $name!

Integer

  • Non-decimal number between -2,147,483,648 and 2,147,483,647


$x = 5985;
$y = -1;

Float

  • Number with a decimal point or a number in exponential form


$x = 10.365;
$y = 7E-10;

Boolean

  • Represents two possible states: TRUE or FALSE
  • Often used in conditional testing


$x = true;
$y = false;

Constants

Constants

  • Constants are like variables except that once they are defined they cannot be changed or undefined.
  • A valid constant name starts with a letter or underscore (no $ sign before the constant name).
  • Constants can store only boolean, integer, float, or string data.


define('PI', 3.14);
echo PI;

Operators and Operations

Arithmetic Operators

  • Follows the basic math rules
Operator Name Example
+ Addition $a + $b
- Subtraction $a - $b
* Multiplication $a * $b
/ Division $a / $b
% Modulus $a % $b
++ Increment $a++
-- Decrement $a--
  • Note: If you apply them to strings, PHP will try to convert the string to a number. If it contains an e or an E, it will be read as being in scientific notation and converted to a float, otherwise, it will be converted to an integer.

Concatenation

  • Use a period to join strings into one


$string1="Hello";
$string2="PHP";
$string3=$string1 . " " . $string2;
echo $string3;

Combined Assignment Operators

Operator Use Equivalent To
+= $a+=$b $a=$a+$b
-= $a-= $b $a=$a-$b
*= $a*= $b $a=$a*$b
/= $a/= $b $a=$a/$b
%= $a%=$b $a=$a%$b
.= $a.= $b $a=$a.$b

Operator precedence

  • Multiplication ( * ) and division ( / ) take precedence over addition (+) and subtraction (–).
  • Use parentheses () to group operations 


$result = ( 5 + 5 ) * 10;
